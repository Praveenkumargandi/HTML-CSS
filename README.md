HTML:
HTML (HyperText Markup Language) is the standard language used to structure web pages. It defines elements like headings, paragraphs, images, links, and forms. It provides the basic skeleton of a website that browsers can read and display.

CSS:
CSS (Cascading Style Sheets) is used to style and design web pages. It controls colors, fonts, layout, spacing, and overall visual appearance. CSS makes a webpage look attractive, responsive, and user-friendly.
